Adicionar o Typescript e as tipagens do react e node
yarn add typescript @types/react @types/node -D

Adicionar import {AppProps} from 'next/app' na pagina _app.tsx = {Component, pageProps}: AppProps

Adicionar o sass (https://sass-lang.com/)
yarn add sass

Adicionar o styles-components (https://styled-components.com/)
yarn add styled-components

Adicionar icone
yarn add react-icons

Adicionar menu collapsavel (https://www.npmjs.com/package/react-collapse)
yarn add react-collpase
yarn add @types/react-collapse

### Instalando o ChrakaUI ###
https://chakra-ui.com/
yarn add @chakra-ui/react @chakra-ui/core @emotion/react @emotion/styled framer-motion
